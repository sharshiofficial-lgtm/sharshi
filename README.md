How to publish on GitHub Pages (quick):
1. Create a GitHub account if you don't have one.
2. Create a new repository named: `sharshi.github.io`
3. Upload all files from this ZIP into the repository root.
4. Commit and push. Wait 1-2 minutes.
5. Your site will be available at: https://sharshi.github.io

Notes:
- Replace the contact email and phone in contact.html.
- The contact form uses Formspree as an example; replace the action URL with your form endpoint or set up email forwarding.
